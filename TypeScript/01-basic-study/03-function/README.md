# 03-function
## js 函数定义
```js
// 函数声明式
function run1() {
  return 'run1';
};

// 匿名函数（函数表达式）
const run2 = function() {
  return "run2";
};

// 构造函数
const run3 = new Function("return 'run3';")
```

## ts 函数定义
1. 函数参数定义
2. 可选参数
在 js 中，函数的实参和形参个数可以不同，ts 中采用可选参数，
注意： 可选参数必须配置到参数的最后面
3. 默认参数
在 es5 中不可设置默认参数， es6 和 ts 中可设置默认参数
4. 剩余参数
三点运算符
5. 函数重载
java: java中方法的重载，指的是两个或两个以上的同名函数，但它们的参数不一样，这时会出现函数重载的情况。
ts:ts中的方法重载，通过为同一个函数提供多个函数类型定义来尝试多种功能的目的。
